## Live Event-Control Checklist

Run this every 30-45 minutes during competition.

| Check | Green | Amber | Red | Action |
|---|---|---|---|---|
| Timetable drift | <10m | 10-20m | >20m | Technical Lead decides compression or cancellation options. |
| Call-room congestion | Flowing | Queue building | Missed calls | Add volunteer / delay affected block. |
| Heat/hydration | No concern | Watch list | Medical concern | Medical Lead decides response. |
| Results backlog | Current | 1 event behind | 2+ events behind | Add runner / pause publication. |
| Field-event delay | <10m | 10-20m | >20m | Reduce warmup, split group, or cut optional event by approval. |
| Throws boundary | Controlled | Needs attention | Breach | Stop throws until controlled. |
| Communications | Approved/current | Update pending | Conflicting messages | One source of truth; Comms Lead resolves. |
| Staffing | Full | Minor gap | Start-gate role gap | Competition Director redeploys or pauses. |

## Results Publication Gate

No result is public until:

- [ ] Event sheet received
- [ ] Data entered
- [ ] Result verified by Results Lead
- [ ] Technical Lead approves correction/dispute status
- [ ] Communications Lead approves publication format
- [ ] Publication log updated
