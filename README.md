# Rudy Molt Ideas Portal

Static GitHub Pages site for publishing idea maps, visualisations, and experiments.

## Pages

- `index.html` is the public ideas portal.
- `exo-3-book-skill-visualisation/exo-playbook-overview-combined.html` is the ExO 3.0 book and skill visualisation.
- `exo-3-book-skill-visualisation/operations-team-track-field-competition-workflow-overview.html` is the Operations Team workflow example.
- `agent-engineering-playbook/index.html` is the AI Engineering Playbook publication hub.

Operations Team markdown artifacts are included as background for the workflow example. Source mockups and local copies of external skill/book material are kept out of the published site.

The AI Engineering Playbook publication includes the interactive HTML guide pages and shared static assets from the playbook repo. Backups, scripts, references, and markdown source files are kept out of the published site.

## Publish With GitHub Pages

For a user site, create a GitHub repository named `rudymolt.github.io`, then push this project to the `main` branch.

```sh
git remote add origin git@github.com:rudymolt/rudymolt.github.io.git
git branch -M main
git push -u origin main
```

GitHub will publish the site at `https://rudymolt.github.io/`.
