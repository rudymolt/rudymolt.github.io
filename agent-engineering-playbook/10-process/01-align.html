AI Engineering Playbook · §10.01

Align Before Code

The cheapest moment to fix a misunderstanding is before code exists. Use the smallest set of alignment tools that settles the product, design, engineering, and vocabulary questions.

Publication · Interactive HTML guide

Which Alignment Tool?

Exit Criteria

  • Every meaningful branch has an answer or explicit out-of-scope decision.
  • Stable vocabulary has been captured or queued for CONTEXT.md.
  • Hard-to-reverse trade-offs have been captured or queued for ADRs.
  • UI-affecting work has an accepted mockup or explicit deferral.
  • The user has confirmed moving forward.
